# RuneHelper
Helping people efficiently build their OSRS character

## Running the app
Clone this repository `git clone git@github.com:theRJMurray/runehelper.git`  
`npm install`
`npm run frontend-install`  
`npm run dev`

## Running tests
`npm install`  
`npm test`
